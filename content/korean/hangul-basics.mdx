# Hangul Basics: Learn the Korean Alphabet

Hangul (한글) is the Korean writing system, created in 1443 by King Sejong the Great. It's considered one of the most logical and easy-to-learn writing systems in the world.

## Why Learn Hangul First?

Before diving into vocabulary or grammar, learning Hangul is essential. Unlike trying to learn Korean through romanization (writing Korean words with English letters), reading Hangul directly will:

- Help you pronounce words correctly
- Allow you to use Korean dictionaries
- Make every future lesson easier to follow

## Basic Vowels (모음)

Korean has 10 basic vowels:

| Hangul | Romanization | Sound |
|--------|-------------|-------|
| ㅏ | a | "ah" as in "father" |
| ㅓ | eo | "uh" as in "sun" |
| ㅗ | o | "oh" as in "go" |
| ㅜ | u | "oo" as in "moon" |
| ㅡ | eu | no English equivalent — say "oo" with lips unrounded |
| ㅣ | i | "ee" as in "see" |
| ㅐ | ae | "eh" as in "bed" |
| ㅔ | e | "eh" as in "bed" (nearly identical to ㅐ in modern Korean) |
| ㅑ | ya | "yah" |
| ㅕ | yeo | "yuh" |

## Basic Consonants (자음)

Korean has 14 basic consonants:

| Hangul | Romanization | Sound |
|--------|-------------|-------|
| ㄱ | g/k | between "g" and "k" |
| ㄴ | n | "n" as in "no" |
| ㄷ | d/t | between "d" and "t" |
| ㄹ | r/l | between "r" and "l" |
| ㅁ | m | "m" as in "me" |
| ㅂ | b/p | between "b" and "p" |
| ㅅ | s | "s" as in "see" |
| ㅇ | ng/silent | silent at start of syllable, "ng" at end |
| ㅈ | j | "j" as in "just" |
| ㅊ | ch | "ch" as in "cheese" |
| ㅋ | k | hard "k" |
| ㅌ | t | hard "t" |
| ㅍ | p | hard "p" |
| ㅎ | h | "h" as in "hello" |

## How Syllable Blocks Work

Korean characters are grouped into **syllable blocks**. Each block contains 2–4 characters arranged in a specific pattern:

1. **Consonant + Vowel**: 나 (na), 고 (go)
2. **Consonant + Vowel + Consonant**: 한 (han), 글 (geul)

The first consonant goes top-left, the vowel goes to the right or below, and any final consonant goes at the bottom.

## Practice

Try reading these common words:

- **한국** (hanguk) — Korea
- **사람** (saram) — person
- **감사합니다** (gamsahamnida) — thank you
